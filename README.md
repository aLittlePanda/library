# library

图书管理系统

##  功能列表

  ###  用户注册 login-register

  ###  登录 login

  ###  用户中心 users

    ####  用户信息修改 users-edit
    ####  用户信息新增 users-add
    ####  用户信息删除 users-delete

  ###  功能部分 common

    ####  左侧导航 common-left

    ####  顶部导航 common-top

  ###  书籍查询 borrow

    #### 书籍借出 books-borrow

  ###  书籍归还 borrow

    #### 书籍归还 books-return

  ###  书籍管理 books

    #### 书籍信息修改 books-edit
    #### 书籍信息新增 books-add
    #### 书籍信息删除 books-delete


##  Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
